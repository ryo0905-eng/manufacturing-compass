# Phase 1.5 Implementation Brief

Phase 1.5 では、Manufacturing Compass の中心機能を「半導体キャリア市場価値診断」に寄せる。

## 作るもの

- 診断開始を主役にしたトップページ
- 半導体キャリア市場価値診断
- 市場価値スコア
- 想定年収レンジ
- 今狙える職種・会社
- 将来のストレッチ目標
- 伸ばすべきスキル・経験・英語
- 今日やること
- 30日、90日、6か月ロードマップ
- 診断結果に応じた転職エージェント CTA
- 診断結果に応じた英語学習 CTA
- 企業ページ・比較ページへの根拠リンク

## 作らないもの

- ログイン
- 個人情報保存
- AI キャリア診断
- 求人応募機能
- 求人データ連携
- Supabase 接続
- 管理画面
- 半導体以外のカテゴリ

## 初期データ方針

静的ルールベースで始める。

必要なデータ:

- 診断質問
- 診断選択肢
- 診断ルール
- 市場価値スコア
- 年収レンジ
- 職種プロファイル
- 企業データ
- アフィリエイト CTA

## 初期ページ範囲

- `/`
- `/diagnosis` または `/career-value-check`
- `/industry-map`
- `/companies`
- `/companies/[slug]`
- `/compare`
- `/compare/[slug]`

既存の `/career-compass` は、診断機能の初期版として扱う。

## UX 方針

- 読むサイトではなく、診断するサービスにする
- 最初にユーザー自身の市場価値を見せる
- 年収レンジは断定せず、目安として表示する
- 伸ばすべき要素を具体化する
- 今日やることを必ず出す
- CTA は診断結果と対応させる

## 技術方針

- Server Components を基本にする
- 診断 UI は Client Component として実装する
- 診断ロジックは静的データから開始する
- 個人情報は保存しない
- SEO メタデータを各ページで定義する
- sitemap に診断ページを含める
