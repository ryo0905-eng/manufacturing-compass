# Phase 1.5 Implementation Brief（履歴）

この文書は、Phase 1.5 で「半導体キャリア市場価値診断」を検討した事実を残すための履歴です。現行要件には使用しません。

- 記録時点: Phase 1.5 検討時
- 履歴化: 2026-07-19
- 現行要件: [`PRD.md`](./PRD.md)
- 現行実装: [`architecture.md`](./architecture.md)

## 当時の検討内容

- `/diagnosis` と独立した結果ページ
- 市場価値スコアと Career Power
- Reward Range としての年収表示
- AI Insight
- 英語学習・資格 CTA の自動出し分け

## 現在採用していない理由

- 現行ルートは `/career-compass` で、結果も同じページ内に表示する
- スコアは人物の市場価値ではなく「転職材料の整理度」として扱う
- 年収は職種別の固定参考帯とし、個人の査定や報酬として扱わない
- AI API、個人情報保存、診断履歴保存を使わない
- CTA は検索意図と相談論点に合う場合だけ表示する

この履歴にある機能を復活させる場合は、過去案をそのまま実装せず、需要、信頼性、プライバシー、費用、運用責任を確認して PRD から更新します。
