# UI Consistency Audit

監査日: 2026-07-21

## 目的と判定基準

公開ページが [`design-system.md`](./design-system.md) の色、文字階層、情報密度、装飾、例外条件に沿っているかを確認した記録です。UIルールの正本は `design-system.md` であり、この文書は現時点の適用状況と移行順を示します。

`src/app/**/page.tsx` の25ルートテンプレートを対象に、ページ構造、適用されるCSS、CSSの上書き順を確認しました。動的ルートはテンプレート単位で判定しています。共有された業界地図の画面も比較対象にしました。2026-07-21に指摘した4ページのCSS移行を完了し、コード上の再監査へ反映しています。実行環境のブラウザ接続を利用できなかったため、全ルートの実表示確認は未実施です。

判定は次の4種類です。

- 準拠: 現在の共通ルールと同種ページの密度に概ね合う
- 承認済み例外: 目的に沿った限定的な例外で、通常ページへ広がっていない
- 一部要調整: 全体は使えるが、一部に旧装飾が残る
- 要移行: ページの中心表現が旧UIで、共通ルールとの差が大きい

## 2026-07-21 修正内容

### 1. `/companies/[slug]`

見出しを32〜36pxへ戻し、青緑のセクション帯、準備ブロック、タグ、概要パネルを中立色へ統合しました。企業固有情報、出典、更新日、準備ポイントの階層は維持しています。

### 2. `/tools`

ヒーローを32〜38pxの見出しへ縮小し、フロー、ロードマップ、カードを白・薄灰・通常罫線へ統一しました。グラデーション、影、ホバー時の浮き上がりを除き、図表の系列色だけを残しています。

### 3. `/career-consultation`

相談前の論点整理という役割は維持し、中央寄せの巨大見出しを通常の文字階層へ変更しました。金色グラデーションは薄灰の相談メモへ置き換えています。

### 4. `/career-agents`

相談テーマ表から装飾グリッドと暗色見出しを除き、含まれる相談テーマを青い選択表示へ変更しました。広告であることを示す開示表現は意味のある例外として維持しています。
