# Design System

## デザイン方針

Manufacturing Compass は、単なる読み物サイトではなく、診断結果を見ながら自分のキャリアの現在地を理解するプロダクトです。

UI は以下を目指す。

- 信頼できる
- 直感的
- 少しワクワクする
- 診断結果が読みやすい
- 数値やスコアが分かりやすい
- アフィリエイトサイトっぽく見えない

## 画面の主役

主役は診断結果。

以下を視覚的に分かりやすくする。

- 市場価値スコア
- 想定年収レンジ
- 今狙える会社
- 将来狙える会社
- 伸ばすべきスキル
- 今日やること

## 色の方向性

- ベース: 白、薄いグレー
- テキスト: 黒に近いニュートラル
- メイン: 信頼感のあるブルーまたはシアン
- 成長・可能性: グリーン
- 年収・市場価値: 控えめなゴールド
- 注意・不確実性: 落ち着いたアンバー

過度なグラデーションや派手な煽り色は避ける。

## タイポグラフィ

- 診断結果の数字は大きく見せる
- 本文は短くする
- 長文説明より、カード、ステップ、チェックリストを使う
- スマホで読みやすい余白を保つ

## 診断 UI

- 質問は1画面に詰め込みすぎない
- 選択肢はカード型にする
- 選択後すぐ結果が変わるとアプリ感が出る
- 結果は「判定」ではなく「現在地」として見せる
- 市場価値スコアは前向きな指標として扱う

## 結果 UI

結果ページは以下の順番で表示する。

1. 市場価値スコア
2. 想定年収レンジ
3. 今狙える会社
4. 伸ばすべき要素
5. 今日やること
6. 30日、90日、6か月ロードマップ
7. CTA

## CTA デザイン

- CTA は診断結果と対応させる
- 「相談する」だけではなく、相談論点を明示する
- 英語学習 CTA は、英語が市場価値に効く理由を示した後に置く
- 広告表記は小さくても明確に出す

## 企業ページデザイン

企業ページは診断結果の裏付けとして扱う。

- 会社概要より、転職者に関係する情報を上に出す
- 年収レンジ、職種、英語、必要経験、準備ポイントを見やすくする
- 企業情報は情報ソースを明記する

## モバイルファースト

- 診断はスマホで完了できることを最優先にする
- 選択肢はタップしやすくする
- CTA は押しやすくする
- 長い表は避け、カード型にする
