# Design System

## デザイン方向性

Manufacturing Compass は、清潔感、余白、信頼感、プロフェッショナルさを重視します。少し Apple 風の静かな上質さを目指しつつ、派手すぎる装飾や強すぎるアフィリエイト感は避けます。

スマホで読みやすく、企業研究や比較を落ち着いて進められる UI にします。

## 色の方向性

- ベース: 白、薄いグレー
- テキスト: 黒に近いニュートラル
- メイン: 信頼感のあるブルーまたはシアン
- アクセント: 控えめなグリーン
- 注意表示: 落ち着いたアンバー

強い原色や過度なグラデーションは避けます。

## タイポグラフィ

- 日本語本文の読みやすさを優先する
- 見出しは短く明確にする
- 行間を広めに取り、長文でも疲れにくくする
- 数値、比較表、会社名が読みやすいサイズを保つ

## カードデザイン

- 角丸は控えめにする
- 影は弱く、境界線と余白で整理する
- 企業カードには会社名、セグメント、主力製品、英語必要度、CTA を載せる
- 情報密度を上げすぎず、比較しやすさを優先する

## ボタンデザイン

- 主要 CTA は目立つが押し売り感を出さない
- セカンダリボタンは控えめにする
- ボタン文言は行動が分かるものにする
- スマホで押しやすい高さを確保する

## CTA デザイン

- 有益な情報の後に自然に配置する
- 「相談する」「近い求人を探す」など、ユーザーの次の行動に寄り添う文言にする
- 広告表記やアフィリエイトであることを適切に示す
- 複数回出す場合も、圧迫感を避ける

## テーブルデザイン

- 企業比較では横スクロールを前提にしすぎない
- 重要な比較軸を固定し、スマホではカード型比較も検討する
- 数値には単位、年度、情報ソースを明記する
- 不明な項目は空欄にせず「公開情報で確認中」などと表現する

## 業界地図デザイン

- ファブレス、ファウンドリ、IDM、製造装置、材料、EDA、OSAT などを整理する
- セグメント間の関係が一目で分かる構造にする
- 初心者にも分かる短い説明を添える
- 企業ロゴの利用は権利に注意する

## 企業比較デザイン

- 会社概要、事業領域、主力製品、売上規模、日本拠点、職種、英語必要度を比較する
- 勝ち負けの表現ではなく、向いている人や準備ポイントを示す
- 今狙える可能性と将来のストレッチ目標を分けて見せる

## キャリア現在地 UI

キャリア現在地 UI は、ユーザーが判定される印象ではなく、前向きに背中を押される印象にします。

- 「不足」よりも「準備ポイント」と表現する
- 「無理」ではなく「今すぐはチャレンジング」と表現する
- 今日からできる行動を必ず提示する
- 半年後・1年後の行動を段階的に示す

## モバイルファースト方針

- 主要ページはスマホ閲覧を前提に設計する
- 比較表、カード、CTA は片手操作しやすくする
- 重要情報をページ上部に集約する
- 長文は見出し、箇条書き、短い段落で読みやすくする
