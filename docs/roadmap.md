# Roadmap

## 7日目標

- Phase 0 の設計ドキュメントを完成させる
- MVP の対象を半導体業界に絞る
- 初期ページ構成、SEO 方針、データモデルを決める
- 初期企業候補 10 社を洗い出す
- Phase 1 で公開する初期企業 3〜5 社を決める
- 小さく公開するための最短ルートを確認する

## 30日目標

- Next.js MVP を作成する
- トップページ、業界地図、企業一覧、企業詳細、比較ページを公開する
- 初期企業 3〜5 社分の情報を整備する
- Google Search Console と Analytics を設定する
- 転職エージェントのアフィリエイト CTA を設置する
- 需要検証のため、検索流入と CTA クリックを測定する

## 3か月目標

- 企業 DB を拡張する
- 公開企業データを 10 社以上に増やす
- セグメントページと比較ページを増やす
- 初心者向け記事を継続的に追加する
- 半導体転職、企業名、職種名、業界用語の SEO 資産を積み上げる
- アフィリエイト導線の位置、文言、比較軸を改善する

## 6か月目標

- キャリア現在地診断を追加する
- 今狙える会社と将来のストレッチ目標企業を分けて提示する
- 半年後・1年後に向けた準備アクションを出せるようにする
- CV 改善のため、ページ別 CTA と内部リンクを見直す
- メール登録やニュースレター導線を検討する

## 1年目標

- 半導体領域で信頼できるキャリア研究サイトとして認知される
- 企業 DB、比較ページ、記事を継続的に更新する
- AI キャリア診断の導入を検討する
- 求人掲載、スポンサー掲載、有料レポートなどの追加収益を検証する
- 将来の転職準備を支援する長期ロードマップ機能を育てる
