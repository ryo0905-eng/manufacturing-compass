# Roadmap

## 7日目標

- サービスの中心を「半導体キャリア市場価値診断」に再定義する
- 既存ドキュメントを診断中心の構成に更新する
- 診断入力項目を決める
- 市場価値スコア、想定年収レンジ、伸ばすべき要素の出力設計を決める
- 転職エージェント CTA と英語学習 CTA の出し分け方針を決める

## 30日目標

- 静的ルールベースの診断 MVP を公開する
- トップページの主導線を診断開始にする
- 診断結果ページを作る
- 市場価値スコアと想定年収レンジを表示する
- 今狙える会社・職種と将来のストレッチ目標を表示する
- 今日やること、30日、90日、6か月ロードマップを表示する
- 転職エージェント CTA と英語学習 CTA を設置する
- Google Search Console と Analytics を設定する

## 3か月目標

- 診断ロジックを改善する
- 職種別の診断結果パターンを増やす
- 年収レンジの根拠となる公開求人・市場情報を整理する
- 企業 DB を10社以上に拡張する
- SEO記事を追加する
- 診断結果から企業詳細、比較、記事への内部リンクを強化する
- CTA の文言、位置、出し分けを改善する

## 6か月目標

- 診断結果の保存やメール送付を検討する
- メール登録導線を追加する
- 英語学習、資格、学習教材のアフィリエイトを検証する
- 有料ロードマップや個別レポートの需要を検証する
- アクセス解析、CV改善、A/Bテストを行う

## 1年目標

- 半導体キャリア市場価値診断として認知される
- AI診断の導入を検討する
- 求人データ連携を検討する
- ユーザーアカウントや診断履歴保存を検討する
- 半導体で検証した後、電子部品、FA、ロボット、自動車、医療機器領域へ拡張する
