# Roadmap

最終更新日: 2026-07-19

具体的な担当作業と完了履歴は [`TASKS.md`](../TASKS.md) で管理します。この文書は、フェーズごとの目的と着手条件だけを扱います。

## 完了した基盤

- Next.js による公開サイト
- Career Compass の静的ルールベース体験
- 業界地図、企業一覧・詳細・比較、キャリア準備
- ガイド、ランキング、SEO メタデータ、構造化データ、sitemap
- 運営者・法務・広告表記ページ
- Vercel Analytics と主要コンバージョンイベント
- 承認済み案件を含む転職エージェント導線
- 実体験記事と半導体技術解説クラスター

## 現在: 信頼性と集客の検証

目的は、新機能を増やすことではなく、公開情報の品質と既存導線の需要を確かめることです。

- Search Console の表示、クエリ、インデックス状態を定期確認する
- 高意図の実体験記事を、取材と匿名化を経て追加する
- 技術解説記事は表示クエリと更新必要性を見て改善する
- 企業ページの個別キャリア情報を優先企業から増やす
- Career Compass の開始、完了、次の行動、相談遷移を計測する
- ASP 成果と外部クリックを照合する
- 提携審査中の案件は承認後に広告表記とリンクを更新する
- 十分な母数が集まるまで CTA の大幅な A/B 最適化を行わない

## 次: コンテンツと導線の改善

着手条件: Search Console とイベント計測で、改善対象となる流入・離脱・相談行動を特定できること。

- 検索意図の重複を避けて記事を追加・統合する
- 企業・職種・工程・記事間の内部リンクを改善する
- 計測データに基づき Career Compass の文言と結果の優先順位を改善する
- 企業キャリア情報の公開対象を増やす
- 必要に応じて記事の短尺動画化を小さく検証する

## 保留中の候補

以下は需要と運用条件を確認するまで実装前提にしません。

- メール登録や結果送付
- 英語学習、資格、教材の追加アフィリエイト
- 有料ロードマップや個別レポート
- AI による補足やキャリア診断
- ユーザーアカウントと履歴保存
- 求人データ連携
- 電子部品、FA、ロボット、自動車、医療機器への拡張

## 保留候補を着手する共通条件

- 既存ユーザー行動から需要を確認できる
- 解く課題と成功指標を PRD に追加できる
- 個人情報、外部送信、費用、運用責任を説明できる
- 信頼性と SEO を損なわず、小さく検証できる
